# KP Astrology App (Antigravity)

This is a high-precision KP (Krishnamurti Padhdhati) Astrology application built with React, TypeScript, and Tailwind CSS.

## Features

- **South Indian Kundli Chart**: Accurate planetary and house cusp positions.
- **KP Significators**: Full 4-Step Theory (Nangu Padi Kurikatigal) implementation.
- **Vimshottari Dasa**: Multi-level (Mahadasa, Bhukti, Antara) calculations with high-precision solar year timing.
- **Nakshatra Metrics**: Detailed Star Lord and Sub Lord calculations.
- **Responsive Design**: Polished UI with Tamil language support for planetary labels.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS 4.
- **Calculations**: `astronomy-engine` for geocentric planetary positions.
- **UI Components**: Shadcn UI, Lucide React, Motion (for animations).

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/lib/astrology.ts`: Core astrological logic (Ayanamsa, Dasa, Significators).
- `src/components/`: Reusable UI components (Chart, Tables, Forms).
- `src/App.tsx`: Main application entry and state management.
